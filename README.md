# LLM Frontend


## Adding Functionality
Include all new functionality in the `src/components` folder, creating a `*.tsx` (typescript) file and a corresponding `*.css` (styling) file.


## Dependencies
- Vite
- React
- Node.js
- Axios (net/http)


## Application Controls
### To Run
```
npm run dev
o <enter>
```

### To Quit
```
q <enter>
```
